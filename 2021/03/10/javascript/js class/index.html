es5 与 es6 类的区别

  1. es5 类定义的方法是可枚举的，es6 类定义的方法不可枚举
  2. es5 构造函数可以作为函数单独执行，es6 类不可以单独执行，必须使用 new 关键字
  3. es6 类不存在变量提升（hoist），这一点与 ES5 完全不同。

类的 get 和 set
