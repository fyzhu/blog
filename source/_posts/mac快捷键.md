### 窗口操作
隐藏 H

关闭 W

最小化 M

全屏 command + control + F

浏览器全屏(窗口全屏的基础上) command + shift + F

vscode 全局搜索 command + shift + F
### 显示隐藏文件、文件夹
点击空白处 然后 command + shift + 。

### 截屏、录屏
command + shift + 5
### chrome 浏览器
control + tab 切换到下一个标签
command + 数字键 切换到指定数字顺序的标签
command + option + ← 切换到左边的标签  
command + option + → 切换到右边的标签
