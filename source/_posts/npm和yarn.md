## 全局操作

npm 加参数 -g

yarn 加参数 global

## 查看npm包

npm list
npm list --depth=0 
npm list --depth --global

yarn list

## 安装npm包

npm install
npm i

yarn add

## 删除npm包

npm uninstall <package>

yarn remove <package>

## 更新npm包

npm install <package>@latest
npm update <package>

yarn add <package>
yarn upgrade <package>
