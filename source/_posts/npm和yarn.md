## 查看本地安装的npm包

npm list
npm list --depth=0 
npm list -g
npm list --depth --global

yarn list
yarn global list

## 安装npm包

npm install
npm install -g
npm i

yarn add
yarn global add

## 删除npm包

npm uninstall
