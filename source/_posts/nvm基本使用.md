查看本地
```
nvm list/ls
```
查看
```
nvm ls-remote
```
安装
```
nvm install <version>
```
切换版本
```
nvm use <version>
```
查看当前版本
```
nvm current
```
设置默认版本
```
nvm alias default <version>
```
