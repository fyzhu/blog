### 对象的属性

属性分两种：数据属性和访问器属性。

#### 数据属性

1. configurable
2. enumerable
3. writable
4. value

#### 访问器属性

1. configurable
2. enumerable
3. get
4. set



访问器属性是不能直接定义的，必须使用 Object.defineProperty() 。
